课标解读

不等式是描述数量大小关系的数学模型，线性规划是优化理论的重要基础。高考中对不等式与线性规划的考查体现了数学的应用价值和工具作用。

核心要求
理解不等式的性质，掌握基本不等式及其应用，会解一元二次不等式、绝对值不等式，理解线性规划的概念，能解决简单的线性规划问题。

考查重点：

  • 不等式的性质与解法
  • 基本不等式及其应用
  • 绝对值不等式的解法
  • 一元二次不等式的解法
  • 线性规划问题的求解
  • 不等式在实际问题中的应用

能力要求：

  • 逻辑推理与不等式证明能力
  • 代数变形与不等式求解能力
  • 几何直观与数形结合能力
  • 数学建模与优化问题解决能力
核心公式：
基本不等式：a² + b² ≥ 2ab
均值不等式：a+b ≥ 2√(ab) (a,b>0)
绝对值不等式：|a| - |b| ≤ |a±b| ≤ |a| + |b|
一元二次不等式：ax² + bx + c > 0
